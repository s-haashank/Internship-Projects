# 🔐 FTP Cracker Tool (Python)

## Description

This tool performs a brute-force attack on FTP servers by trying multiple usernames and passwords from wordlists or generating them dynamically.

**⚠️ Educational Use Only!**  
This project is intended for ethical hacking, penetration testing, and cybersecurity learning purposes **on systems you own or have permission to test**.

---

## 🚀 Features

- Multi-threaded FTP brute force
- Supports username and password wordlists
- Option to generate passwords on the fly
- Easy-to-use command-line interface
